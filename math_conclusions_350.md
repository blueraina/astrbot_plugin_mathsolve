# 数学结论整理（350条）

说明：按 7 个学科分别整理，每科 50 条，共 350 条。每条尽量控制为一两句话，偏重常用二级、三级结论、易错点和做题实用结论。

## 1. 常微分方程

1. 一阶线性方程 y' + p(x)y = q(x) 的标准做法是乘积分因子 μ(x) = exp(∫p(x)dx)。易错点：μ 要乘整个方程，最后再整体积分。
2. 可分离变量方程要先把 y 项和 x 项分到两边，再分别积分。易错点：分离时不能把可能为 0 的因子随便约掉，常值解要单独补上。
3. 形如 y' = F(y/x) 的一阶齐次方程，通常设 y = ux 或 x = vy 来降阶。易错点：代换后别忘了 y' = u + xu'。
4. Bernoulli 方程 y' + p(x)y = q(x)y^n，常设 z = y^(1-n) 化成线性方程。易错点：n = 1 时它本身就是线性方程，不要重复变形。
5. 恰当方程 M(x,y)dx + N(x,y)dy = 0 的核心判据是 M_y = N_x。易错点：这个判据只在所讨论区域内函数足够光滑时才好用。
6. 求恰当方程势函数时，可以先对 M 关于 x 积分，再用 N 去校正“只与 y 有关”的项。易错点：校正函数是单变量函数，不是常数。
7. 自治方程 y' = f(y) 的平衡点来自 f(y) = 0。判断稳定性最直接的方法是看平衡点两侧 f(y) 的符号变化。
8. 连续不代表唯一，Lipschitz 条件才更接近唯一性保证。典型易错点是把“有解”误当成“唯一解”。
9. 二阶线性齐次方程的解空间是二维的，只要找到两个线性无关解就得到了通解。易错点：两个解不是“看起来不同”就一定线性无关。
10. 常系数齐次方程先写特征方程。不同实根对应指数解，重根要乘 x，复根要写成 e^(αx)(C1 cosβx + C2 sinβx)。
11. 特征方程有重根 r 时，对应解不是只有 e^(rx)，还要补 xe^(rx)、x^2e^(rx) 等。漏乘 x 是最常见错误。
12. 对于二阶常系数非齐次方程，通解一定是“齐次通解 + 一个特解”。易错点：特解不是唯一的，但通解形式必须完整。
13. 待定系数法只适合右端是指数、三角、多项式及其有限乘积这类函数。遇到变系数或奇怪右端时不要硬套。
14. 右端函数若与齐次解“撞车”，待定特解要乘 x 的适当幂次。判断幂次的原则是补到与齐次解线性无关为止。
15. 参数变易法本质上总能做，但计算量往往比待定系数法大。它要求先知道对应齐次方程的一组基解。
16. 对二阶线性方程，Wronskian 在某点不为 0，则这一组解在线性意义下独立。易错点：只在一个点检验非零即可，不必处处算。
17. Abel 公式说明 Wronskian 满足一阶线性方程，因此要么恒为 0，要么处处不为 0。这个结论在线性方程中非常省计算。
18. 线性初值问题只要系数函数连续，通常就有唯一解。边值问题却不一定，有时无解，有时多解。
19. 已知一个非零解 y1 后，降阶法常设 y = v(x)y1 来求第二个解。易错点：代换后先化简，再把 v'' 项降掉。
20. Euler-Cauchy 方程 x^2y'' + axy' + by = 0 常试 y = x^m。它像“变系数版常系数方程”，但只适合 x 同次幂结构。
21. 对自治二阶方程 y'' = f(y)，把两边乘 y' 往往能得到第一积分。这个技巧相当于把方程改写成“能量守恒”形式。
22. 相线法适合一维自治方程，不需要显式解也能判断长期行为。重点是箭头方向，而不是只看平衡点个数。
23. 最大存在区间由解本身和方程系数的奇点共同决定。初值给定后，解不一定能延到整个实轴。
24. 方程 y' = y^2 的解会在有限时间爆破，这说明“初值问题有唯一解”不等于“解全局存在”。这类点很容易被忽视。
25. 线性系统 X' = AX 的求解核心是矩阵指数 e^(At)。当 A 可对角化时，e^(At) 的计算会明显简化。
26. 线性系统若 A 有 n 个线性无关特征向量，就能直接写出 n 个指数型解。易错点：特征值够多不代表特征向量一定够多。
27. 当矩阵出现 Jordan 块时，解里会出现 t e^(λt)、t^2 e^(λt) 这类项。它和标量方程重根乘 x 的现象是同一逻辑。
28. 二维线性系统平衡点分类可先看迹 tr(A) 和行列式 det(A)。det(A) < 0 必是鞍点，这是最快判断之一。
29. 二维线性系统里，tr(A)^2 - 4det(A) 决定特征值是否实。别把“判别式正负”与“稳定性”混为一谈，稳定还要看实部。
30. 对常系数系统，只要所有特征值实部都小于 0，零解就是渐近稳定。只要有一个特征值实部大于 0，零解就不稳定。
31. 若所有特征值实部都大于 0，则解整体远离平衡点。若既有正实部又有负实部，典型就是鞍型行为。
32. 变参数公式 X(t) = Φ(t)[c + ∫Φ^(-1)(s)f(s)ds] 对线性非齐次系统非常通用。易错点：积分里的变量要用中间变量 s，Φ 也必须是基础解矩阵。
33. Liouville 公式给出 det Φ(t) = det Φ(t0) exp(∫tr A(t)dt)。它能快速判断基础解矩阵是否始终可逆。
34. 简谐振子 y'' + ω^2y = 0 的解是正弦余弦组合，对应守恒量常是 y'^2 + ω^2y^2。看到这种结构要优先想到“能量法”。
35. 阻尼项 c y' 会消耗能量，所以 c > 0 时振幅通常衰减。易错点：衰减不代表一定不过零，也不代表一定单调。
36. 受迫振动若外频率接近系统固有频率，会出现共振现象。严格共振时，特解常要乘 t。
37. 对非线性系统，线性化只在平衡点附近有效。离开平衡点太远时，线性近似结论可能完全失真。
38. Jacobian 线性化法里，若平衡点的特征值实部都不为 0，则局部行为通常由线性部分决定。若出现纯虚特征值，就不能草率下结论。
39. Laplace 变换特别适合处理分段输入、脉冲输入和带初值的问题。易错点：变换时初值项不能漏写。
40. 单位阶跃函数 u(t-a) 在 Laplace 变换中通常伴随平移公式出现。关键不是把 t 换成 t-a，而是把函数整体写成延迟形式。
41. 卷积公式把“乘法变积分”变成“积分变乘法”，对求非齐次线性方程特解很有效。易错点：卷积上下限通常是 0 到 t。
42. 幂级数法通常在普通点附近展开，代入后比较同次幂系数得到递推。易错点：求导后指标经常要平移，最容易抄错。
43. 正则奇点附近常用 Frobenius 法，试解写成 x^rΣa_nx^n。根的差是整数时，第二个解的形式要格外小心。
44. 比较定理常用于证明解的上下界或单调性。使用时要先确保比较的两个方程和初值顺序都满足条件。
45. Gronwall 不等式是估计解对初值、参数依赖性的常用工具。很多“稳定依赖”结论最后都落到它身上。
46. Picard 迭代既能证明存在唯一，也给出近似解思路。易错点：它依赖压缩映射思想，不是任何方程都能直接迭代成功。
47. 非自治方程中“平衡点”概念要谨慎，因为右端显含 x 时常值函数未必是解。不要把自治方程的图像直觉生搬硬套。
48. 边值问题常可用打靶法转成初值问题，再通过调参数满足边界条件。它的难点不在“解初值”，而在“调到边界”。
49. 解的延拓常依赖“只要解保持有界且系数没炸，就能继续延下去”这类思想。真正阻止延拓的往往是爆破或碰到奇点。
50. 做常微分方程题时，先辨认方程类型往往比硬算更重要。很多题目一旦类型看对，后面的计算只是标准流程。

## 2. 概率论与数理统计

1. 概率加法公式 P(A∪B) = P(A) + P(B) - P(AB) 是最常用的修正公式。易错点：两个事件不互斥时，交集不能漏减。
2. 互斥不等于独立。只要 A、B 都有正概率，互斥反而通常意味着它们不独立。
3. 条件概率 P(A|B) 的前提是 P(B) > 0。分母为 0 时这个条件概率本身就没有定义。
4. 全概率公式适合“按原因分类”拆事件。Bayes 公式适合“已知结果反推原因”。
5. Bayes 公式里后验概率与“先验 × 似然”成正比。易错点：别忘了再除以总概率作归一化。
6. 随机变量相等、同分布、独立是三个不同层次的概念。做题时不要把“同分布”误当成“取值一模一样”。
7. 求 E[g(X)] 时，优先直接按分布求和或积分，不一定要先求 g(X) 的分布。这个技巧常能省很多步骤。
8. 期望具有线性性，E(aX + bY + c) = aE(X) + bE(Y) + c。这个结论不要求 X、Y 独立。
9. 方差满足 Var(X) = E(X^2) - [E(X)]^2。易错点：先平方再取期望，和先取期望再平方不是一回事。
10. Var(aX + b) = a^2Var(X)，平移不改方差，缩放会按平方放大。很多计算题都靠这一步快速标准化。
11. 若 X、Y 独立，则 E(XY) = E(X)E(Y)。反过来不一定成立，零协方差也通常不推出独立。
12. 协方差 Cov(X,Y) = E(XY) - E(X)E(Y)，它反映的是线性相关而不是全部相关。非线性关系完全可能使协方差为 0。
13. 相关系数的取值一定在 [-1,1]。当某个变量方差为 0 时，相关系数没有定义。
14. 指示变量法非常高效，I_A 的期望就是 P(A)。很多“计数型随机变量”的期望都能拆成若干指示变量求和。
15. 离散型随机变量的分布列概率和必须为 1，连续型密度积分必须为 1。易错点：密度函数本身可以大于 1。
16. 连续型随机变量满足 P(X = a) = 0，但这不等于事件“X = a”不可能。它只是说明单点没有概率质量。
17. 分布函数 F(x) 一定右连续、单调不减，且极限满足左端到 0、右端到 1。做分布函数题时先查这几个性质。
18. 泊松分布常用于“单位时间或单位区域内稀有事件次数”。当二项分布满足 n 大、p 小、np 适中时，常可用泊松近似。
19. 二项分布的均值是 np，方差是 np(1-p)。最常见错误是把方差误写成 np。
20. 几何分布和指数分布都具有无记忆性。它们是离散和连续情形下少见的“记忆无关”模型。
21. 正态分布题先想到标准化 Z = (X-μ)/σ。表格查值也基本都要先转成标准正态。
22. 独立正态变量的线性组合仍是正态变量。只要能写成线性组合，均值和方差几乎都能直接算。
23. 独立泊松变量之和仍是泊松变量，参数直接相加。这个结论在排队论和计数模型里特别高频。
24. 二项分布近似正态时，通常要求 np 和 n(1-p) 都不太小。做连续近似时最好加连续性修正。
25. 大数定律说明样本均值会稳定到总体均值附近。它解决的是“会不会靠近”，不是“靠近得多快”。
26. 中心极限定理说明标准化后的和或均值近似服从正态。易错点：先减均值再除标准差，标准化步骤别漏。
27. 依分布收敛、依概率收敛、几乎处处收敛强弱不同。常见关系是“几乎处处收敛 ⇒ 依概率收敛 ⇒ 依分布收敛”。
28. 矩母函数若在 0 邻域内存在，常能唯一确定分布。它在证明极限定理和求矩时都很方便。
29. 联合密度求边缘密度时，要先把支持区域画清楚。很多积分上下限错，不是不会算，而是区域没看明白。
30. 条件密度不是把联合密度直接“代进去”就完事，必须除以相应边缘密度。分母少写是典型低级错误。
31. 样本均值是总体均值的无偏估计，且在独立同分布下 Var(X̄) = σ^2/n。样本量翻倍，方差只降到原来的一半。
32. 样本方差用 1/(n-1) 而不是 1/n，是为了在独立同分布情形下保持无偏。考试里这个分母特别容易写错。
33. 无偏性、相合性、有效性不是同一个概念。一个估计量可以有偏但相合，也可以无偏但方差很大。
34. 极大似然估计的一个好处是不变性：θ 的极大似然估计代入 g，通常就得到 g(θ) 的极大似然估计。这个结论在变参数时很省事。
35. 对数似然与似然在最大点上等价，因为对数函数单调递增。实际计算时几乎总是先取对数再求导。
36. 充分统计量的判断常用 Neyman-Fisher 因子分解定理。找到充分统计量后，很多推断问题会明显简化。
37. 置信区间的随机性在区间，不在参数。95% 置信区间不是“参数有 95% 概率落在这个已算出的区间里”。
38. p 值不是“原假设为真的概率”，它是在原假设成立时，观察到当前或更极端数据的概率。这个概念最容易被误解。
39. “未拒绝原假设”不等于“接受原假设”。更多时候它只表示证据还不够强。
40. 显著性水平 α 控制第一类错误概率，检验功效则和第二类错误有关。别把“显著”误解成“差异很大”。
41. 双侧检验和单侧检验的拒绝域不同，p 值也不同。做题时方向一旦选错，结论经常全错。
42. 当总体正态、方差未知且样本量较小时，关于均值的统计量通常走 t 分布而不是正态分布。自由度一般是 n-1。
43. 卡方统计量常来自“标准正态平方和”或“方差型问题”。自由度的扣减常与估计参数个数有关。
44. F 分布是两个独立卡方变量各自除以自由度后的比值。方差分析和回归显著性检验里它很常见。
45. 最小二乘估计的几何本质是把观测向量正交投影到模型空间。残差与设计矩阵的每一列都正交。
46. 回归中相关不代表因果，R^2 大也不代表模型一定合理。很多伪回归恰恰就输在这里。
47. 条件期望 E(X|Y) 是关于 Y 的函数，可看成“已知 Y 后对 X 的最优平方损失预测”。它不是一个常数，除非进一步取总期望。
48. 全期望公式 E[E(X|Y)] = E(X) 非常实用，尤其适合分层建模。很多复杂随机变量都能先条件化再求期望。
49. Jensen 不等式的方向由凸凹性决定：凸函数满足 φ(E(X)) ≤ E[φ(X)]。做题时最怕把不等号方向写反。
50. 做概率统计题时，先分清“分布问题、数字特征问题、推断问题”三类再下手。很多错误并不是不会做，而是一开始把题型认错了。

## 3. 微分几何

1. 正则曲线的基本要求是参数导数不为 0，也就是 r'(t) ≠ 0。只有这样切向量才有意义，曲线才不会“停住”。
2. 用弧长 s 作参数时，曲线满足 |r'(s)| = 1。很多公式在弧长参数下都会明显变简。
3. 同一条几何曲线可以有很多参数化方式，但曲率、挠率等几何量不依赖于正常的重参数。易错点：带符号的量可能会因方向改变而变号。
4. 平面曲线曲率常用公式是 κ = |x'y'' - y'x''| / (x'^2 + y'^2)^(3/2)。只要不是弧长参数，这个公式比直接算 T' 更稳。
5. 对单位速度曲线，曲率就是 κ = |T'(s)|。这说明曲率本质上衡量的是切向量变化得有多快。
6. 直线的曲率恒为 0，圆半径为 R 时曲率恒为 1/R。看到常曲率平面曲线，优先想到圆。
7. 平面曲线的带符号曲率与取向有关，反向走曲线时符号会变。若只关心弯曲程度，通常看绝对值曲率。
8. 只要 κ > 0，就能定义主法向量 N = T'/|T'|。当曲率为 0 时，法向方向并不稳定，别硬套 Frenet 公式。
9. 空间曲线若挠率 τ 恒为 0，并且曲率不恒为 0，则曲线落在某个平面内。这个结论常用来判定“是否是平面曲线”。
10. 圆螺线的曲率和挠率都是常数。反过来，曲率和挠率都为正常数的曲线通常就是某种螺线。
11. Frenet 标架 T、N、B 需要曲线足够光滑且 κ > 0。很多公式默认这些前提，使用前最好先检查。
12. Frenet 公式把曲线的局部几何压缩进 κ 和 τ 两个量里。记忆上可以抓住“切向量变化朝法向，副法向变化反映扭转”。
13. 曲线重新参数化后，速度会变，但几何轨迹不会变。做题时要分清“参数速度变化”与“几何形状变化”。
14. 正则曲面片要求 r_u × r_v ≠ 0。它保证 u、v 两个方向真的张成一个二维切平面。
15. 曲面的切平面由 r_u 和 r_v 张成，法向量常取 r_u × r_v。易错点：法向量方向有两个，取向变了会影响带符号曲率。
16. 第一基本形式 I = E du^2 + 2F dudv + G dv^2 负责长度和角度计算。它是曲面上的“度量”。
17. 正则曲面上必须有 EG - F^2 > 0。这个量其实就是 |r_u × r_v|^2，不能为 0。
18. 面积元是 √(EG - F^2) dudv。看到求曲面积分面积时，这个因子几乎必出现。
19. 曲面法向量单位化后才适合进入第二基本形式和曲率公式。漏掉单位化会让 L、M、N 整体倍率出错。
20. 第二基本形式 II = L du^2 + 2M dudv + N dv^2 反映曲面相对法向的弯曲。第一基本形式管“内在度量”，第二基本形式管“外在弯曲”。
21. 法曲率可以写成 κ_n = II/I，其中 I、II 都沿同一切向方向取值。这个公式说明法曲率依赖方向选择。
22. 主曲率是法曲率的最大值和最小值，也就是形算子的特征值。主方向则是达到极值的方向。
23. Euler 公式说明任意方向法曲率由两个主曲率插值得到。实际做题时，先找主曲率往往比直接算任意方向更省力。
24. 平均曲率 H = (k1 + k2)/2，高斯曲率 K = k1k2。一个反映“平均弯曲”，一个反映“内在弯曲”。
25. K > 0 的点叫椭圆点，K < 0 的点叫双曲点，K = 0 的点叫抛物点。不同符号对应完全不同的局部形状。
26. 平面上 k1 = k2 = 0，所以 H = 0、K = 0。它是最平的曲面，也是很多公式的基准情形。
27. 半径为 R 的球面满足 k1 = k2 = 1/R，因此 K = 1/R^2。球面是最典型的常正曲率曲面。
28. 圆柱面有一个主曲率为 0，另一个一般不为 0，所以 K = 0 但 H 通常不为 0。易错点：K = 0 不代表曲面就是平面。
29. 曲面上一条曲线是测地线，当且仅当它的测地曲率为 0。也就是说，它“只按曲面法向弯，不在曲面内侧拐弯”。
30. 平面上的测地线就是直线，球面上的测地线是大圆。小圆通常不是测地线，这个判断很高频。
31. 测地线局部最短，但不一定全局最短。超过某些长度后，同一条测地线可能不再是最短路。
32. 测地线方程是二阶常微分方程，解由初始点和初始切向量局部唯一确定。它和常微分方程初值理论联系很紧。
33. Christoffel 符号不是张量，它会随坐标改变。易错点：别把“某坐标系里为 0”误解成“几何上恒等于 0”。
34. 法坐标能让 Christoffel 符号在某一点消失，但通常只能在该点消失。邻域里它们一般还会重新出现。
35. 等距映射保持第一基本形式，因此保持长度、角度和面积元。高斯曲率也会随等距保持。
36. 高斯绝妙定理说明 K 只由第一基本形式决定。也就是说，高斯曲率是内在量，不必借助外部空间去定义。
37. 反过来，平均曲率依赖法向选择，改换取向时 H 会变号。K 不会变号，这是两者很大的差别。
38. 若 k1 = k2，则该点是脐点，所有方向都是主方向。球面上每一点都是脐点。
39. 当 k1 ≠ k2 时，主方向彼此正交。这个结论在正交坐标构造里很常用。
40. 渐近方向满足法曲率为 0。椭圆点没有实渐近方向，双曲点通常有两条。
41. 线曲率、法曲率、测地曲率之间满足分解关系：空间弯曲可以拆成“朝法向弯”和“沿曲面内弯”。做曲线在曲面上的题时，这个分解很关键。
42. 曲面的坐标曲线正交当且仅当 F = 0。很多旋转曲面和共形参数都在追求这个简化。
43. 旋转曲面的经线是测地线，纬线是否是测地线要看半径函数。球面上只有赤道是典型纬线测地线。
44. 旋转曲面的测地线满足 Clairaut 关系：r(s)sinθ 保持常数。它是判断测地线能否逼近极点的利器。
45. 高斯映射把曲面法向送到单位球面，它的微分与形算子只差一个符号。很多曲率公式本质上都来自这里。
46. 闭凸平面曲线的总曲率是 2π。这个结论说明“绕一圈以后切向量也刚好转一圈”。
47. 球面三角形的内角和大于 π，超出的部分和面积成正比。平面几何直觉在球面上不能直接照搬。
48. 最小曲面满足 H = 0，但 K 不一定为 0。易错点：最小曲面说的是平均曲率，不是高斯曲率。
49. 曲面上的最短路问题通常先转成测地线问题，再结合边界或对称性筛选。不是每条测地线都是题目要的最短线。
50. 做微分几何题时，先区分“内在量”和“外在量”通常最重要。第一基本形式、高斯曲率偏内在；法向、第二基本形式、平均曲率更偏外在。

## 4. 复变函数

1. 复可导比实可导强得多，在一点复可导已经隐含两个方向上的强约束。不要把“把 z 看成两个实变量可导”误当成“复可导”。
2. 若 f = u + iv 在区域内可导且一阶偏导连续，则满足 Cauchy-Riemann 方程 u_x = v_y，u_y = -v_x。做题时这通常是第一道筛子。
3. Cauchy-Riemann 方程加偏导连续通常也能推出全纯。易错点：只在一点满足方程，并不自动推出在该点复可导。
4. 全纯函数在区域内其实是解析函数，也就是能展开成幂级数。复分析里“可导一次”和“可导无穷次”之间几乎没有间隔。
5. 幂级数的收敛半径由“离展开中心最近的奇点”决定。找奇点位置常比死用根值法更快。
6. 幂级数在收敛圆内可以逐项求导、逐项积分，收敛半径不变。这个结论在求级数和函数展开时非常常用。
7. 全纯函数的零点若不是恒等于 0，就只能是孤立的。零点聚起来通常意味着整个函数都被锁死了。
8. 恒等定理说明两个全纯函数若在有聚点的集合上相等，则在连通区域内处处相等。它是“局部信息决定整体”的经典结论。
9. 最大模原理说非常值全纯函数在区域内部取不到模的最大值。遇到“内部达到最大值”的题，先想到常值结论。
10. 开映射定理说明非常值全纯函数把开集映成开集。它解释了为什么全纯函数局部不会把二维区域压扁成曲线段。
11. Liouville 定理说明有界整函数必为常数。很多题最后都靠“先证有界，再用 Liouville”收尾。
12. 代数学基本定理可以由 Liouville 定理推出：非常数复系数多项式一定有复根。复分析给了它一个非常短的证明。
13. 在单连通区域内，全纯函数沿闭曲线积分为 0。易错点：区域有洞时，不能机械套用这个结论。
14. Cauchy 积分公式是复分析的核心：函数值可由边界积分恢复。它比“闭路积分为 0”更强，也更实用。
15. Cauchy 公式对高阶导数同样成立，因此全纯函数高阶导数都能由边界控制。很多估计题都从这里出发。
16. Cauchy 估计说明导数大小受边界最大模控制。它常用来证明 Liouville 定理和级数收敛性质。
17. Morera 定理提供了一个逆向判别：若连续函数沿任意闭曲线积分为 0，则它全纯。它适合处理极限函数是否全纯的问题。
18. 若函数在孤立点附近有界，则该点是可去奇点。可去奇点最实用的识别方式就是“看有没有爆掉”。
19. 若 |f(z)| 在某点附近像 1/|z-z0|^m 这样发散到有限阶，则 z0 往往是极点。极点的本质是 Laurent 主部只有有限项。
20. 可去奇点、极点、本性奇点是孤立奇点的三种基本类型。判断时最稳的方法是看 Laurent 展开的负幂部分。
21. 留数就是 Laurent 展开里 (z-z0)^(-1) 的系数。只要能抓住这一个系数，积分常常立刻结束。
22. 简单极点的留数可直接用 Res(f,z0) = lim(z→z0)(z-z0)f(z)。这是最省力的公式之一。
23. m 阶极点的留数可以用求导公式算，但实战中先尝试约分、部分分式或对数求导往往更快。公式会用就好，不必每次硬背展开。
24. 留数定理把围道积分变成“2πi × 内部留数和”。做有理函数积分题时，先定位所有奇点比先积分更重要。
25. 无穷远点也是一个可讨论的奇点，常通过研究 g(w) = f(1/w) 在 w = 0 的性质来判断。很多整体性质在“加上无穷远点”后更统一。
26. 一个有理函数在全平面及无穷远点都无奇性时，它必为常数。这个观点很适合快速判定函数形状。
27. 辐角原理把零点和极点个数转成边界上的辐角变化。做“数根”问题时，它非常高效。
28. Rouché 定理适合比较两个函数在边界上的大小，从而判断区域内零点个数是否相同。核心是“主导项不变，零点个数不变”。
29. Schwarz 引理适用于单位圆盘内全纯、且 f(0)=0、|f|≤1 的函数。它会同时给出 |f(z)| 和 |f'(0)| 的 sharp 估计。
30. Möbius 变换把广义圆映成广义圆，并保持交比。处理半平面、圆盘之间的双射时，它几乎是标准工具。
31. 线与圆在复平面里常统一看成广义圆。很多映射题若能换成这个视角，结构会简单很多。
32. 有原函数就有路径无关性，闭路积分自然为 0。在星形区域或单连通区域里，全纯函数通常容易构造原函数。
33. Laurent 展开不是在圆盘里，而是在环域里展开。遇到“中间挖空”的区域时，它比 Taylor 展开更合适。
34. 本性奇点附近函数行为极不稳定，Casorati-Weierstrass 定理说明它在任意邻域内值域稠密。做题时见到 e^(1/z) 之类要高度警惕。
35. 若全纯函数模长恒定，则它必为常数。因为模长不能在内部达到严格极值又处处保持不变。
36. 若整函数的实部有上界或下界，则它也是常数。常见做法是套到 e^f 或利用调和函数的最大值原理。
37. 调和函数满足平均值性质和最大值原理。很多复分析结论其实可以先在调和函数层面理解。
38. 在单连通区域内，调和函数通常可以找到调和共轭，从而组成全纯函数。易错点：区域有洞时，共轭函数未必全局存在。
39. 复对数、平方根等多值函数想变成单值，通常要选支路并切开分支线。分支切口不同，得到的“主值”也不同。
40. 解析延拓一旦存在就是唯一的。也就是说，同一个局部幂级数不可能沿两条路延成两个不同的全纯函数而又在重叠处兼容。
41. 局部逆函数定理在复分析中更强：若 f'(z0) ≠ 0，则 f 在 z0 附近共形且可逆。导数不为 0 是局部保角的关键。
42. 共形映射要求全纯且导数不为 0。易错点：常值函数也全纯，但绝不是共形映射。
43. 反射原理适合把边界上取实值或满足对称条件的全纯函数延拓到另一侧。它在上半平面问题里特别常见。
44. 用留数法算实积分时，围道选择比残差计算更重要。先看 integrand 的衰减性、奇点分布和是否需要绕开实轴奇点。
45. Jordan 引理适合处理带 e^(iaz) 因子的半圆积分。a 的符号决定该往上半平面还是下半平面闭合。
46. 对数导数 f'/f 的极点结构能同时记录零点和极点。它是辐角原理、Rouché 定理背后的统一语言之一。
47. 一列全纯函数若在每个紧集上一致收敛，则极限仍全纯。很多构造题和级数题都要靠这个稳定性结论。
48. 全纯函数列若在每个紧集上一致收敛，则导函数列也在每个紧集上一致收敛到极限函数的导数。这个结论在幂级数和正规族题目里很常见。
49. Poisson 积分公式能用圆周边值重建圆盘内的调和函数。它是圆盘 Dirichlet 问题的标准显式解。
50. 做复变函数题时，先判断“局部用级数，整体用积分，计数用辐角，实积分用留数”往往最有效。方法选对，后面才会顺。

## 5. 实变函数

1. 点态收敛只保证每个点最终收敛，不保证收敛速度在全区间统一。很多“能不能交换极限与运算”的问题，点态收敛远远不够。
2. 一致收敛比点态收敛强，它允许把 sup|f_n-f| 压到 0。只要能证一致收敛，很多连续性和积分交换都会变得安全。
3. 连续函数列一致收敛到的极限函数仍连续。易错点：若只有点态收敛，极限函数完全可能不连续。
4. 若 f_n 可导、f_n' 一致收敛，且某点 f_n(x0) 收敛，则 f_n 一致收敛到某函数 f，且 f' = lim f_n'。这个定理用来交换“极限与求导”特别常见。
5. 单调收敛定理适用于非负可测函数递增列，结论是积分极限等于极限积分。关键条件是“非负”和“单调递增”。
6. Fatou 引理给出 ∫liminf f_n ≤ liminf∫f_n，方向不能写反。它常被看作主控收敛定理的“保底版”。
7. 主控收敛定理要求 f_n 几乎处处收敛且有一个可积函数 g 主控 |f_n|。缺“主控”时，极限与积分往往不能交换。
8. 可积主控函数必须与 n 无关。若每个 f_n 都各自有主控，但主控本身越来越大，主控收敛定理仍可能失效。
9. 绝对可积一定可积，但可积不一定绝对可积。处理广义积分和级数时，这个层次差别非常关键。
10. 在 Lebesgue 理论里，几乎处处相等的两个函数有相同积分。易错点：这不代表它们逐点完全一样。
11. 零测集在积分中通常可以忽略，但在连续性、点值、上确界等问题里未必能忽略。要分清“a.e.”和“处处”的区别。
12. 有限或可数集一定是零测集，但零测集不一定可数。Cantor 集就是常见的反例模板。
13. 连续函数在紧集上一定一致连续且有界。很多题目只要先确认定义域是紧集，后面立刻省掉很多技术细节。
14. Heine-Borel 定理只在 R^n 中成立：闭且有界等价于紧。到无限维空间里，这个判断会失效。
15. 紧集上的连续函数一定能取到最大值和最小值。定义域不紧时，连续函数完全可能有界却取不到上确界。
16. 连续映射把紧集映成紧集，也把连通集映成连通集。做像集结构题时，这两个结论非常高频。
17. R 上的连通子集一定是区间。遇到“连通却看起来零碎”的集合，十有八九有问题。
18. 可测非负函数总能由一列简单函数从下逼近。Lebesgue 积分的定义和很多逼近论证都建立在这一步上。
19. Lusin 定理说明可测函数在大部分区域上“几乎像连续函数”。Egorov 定理则说明在有限测度集上，a.e. 收敛可升级为“除去小集合后一致收敛”。
20. Fubini 定理与 Tonelli 定理都用于交换二重积分顺序。Tonelli 偏向非负函数，Fubini 偏向绝对可积函数。
21. 交换积分次序前先检查非负或绝对可积，这一步不能省。积分区域画错比算错更常见。
22. Riemann 可积的充要条件是函数有界且不连续点集的 Lebesgue 测度为 0。这个结论连接了 Riemann 积分和测度论。
23. 单调函数在闭区间上一定 Riemann 可积，因为它的不连续点至多可数。单调不代表连续，但对积分已经够用了。
24. 单调函数几乎处处可导，但导数未必处处存在。别把“单调”误解成“可导性很好”。
25. 绝对连续函数一定一致连续、变差有界且几乎处处可导。它是“可积导数恢复原函数”的正确函数类。
26. 对绝对连续函数 f，有 f(x) = f(a) + ∫[a,x] f'(t)dt。这个版本才是 Lebesgue 框架下最稳的微积分基本定理。
27. 若绝对连续函数满足 f' = 0 几乎处处，则 f 为常数。易错点：若只知道连续且 a.e. 导数为 0，这个结论一般不成立。
28. 有界变差函数可以写成两个单调函数之差。很多看似复杂的函数，先证 BV 再分解会简单很多。
29. 一列函数在 L1 收敛，不一定逐点收敛；逐点收敛也不一定 L1 收敛。不同收敛方式之间不能随意替换。
30. 在有限测度空间中，几乎处处收敛能推出依测度收敛。反过来一般不成立，但可从依测度收敛中抽出 a.e. 收敛子列。
31. 在有限测度空间中，Lq 常嵌入 Lp（q > p）。测度无限时，这个包含关系往往失效，不能机械套用。
32. Hölder 不等式是处理乘积可积性的基本工具，Minkowski 不等式则是 Lp 空间的三角不等式核心。做 Lp 估计题时这两条几乎是标配。
33. Jensen 不等式的方向取决于函数是凸还是凹。平均先做还是后做，结果常常由凸性决定。
34. 稠密和大测度是两回事。有理数在 R 中稠密，但测度却为 0。
35. 零测和第一类集合也是两回事。一个集合可以测度很小却拓扑上很“大”，反过来也一样。
36. Baire 类别定理说明完备度量空间不能写成可数个稀疏集的并。它常用来构造“处处有”的病态对象。
37. 压缩映射原理给出完备空间中不动点的存在唯一性。很多积分方程、微分方程的存在唯一，本质上都靠它。
38. Cauchy 列在完备空间里一定收敛，不完备空间里则未必。证明极限存在时，先确认空间完备往往是关键一步。
39. 连续函数在开区间上不一定一致连续，比如 1/x 在 (0,1) 上。想用一致连续，最好先找紧性或额外估计。
40. 导数存在不代表导数连续，连续也不代表可导。基础分析里很多反例都来自这两个“不能反推”。
41. Darboux 定理说明导函数虽可很不连续，但仍有介值性。也就是说，导数不能跳跃。
42. 若一列可积函数仅仅点态有界，仍不足以交换极限与积分。真正有用的是统一主控、单调性或一致可积性。
43. 连续映射的零测像不一定仍是零测，但 Lipschitz 映射会把零测集送到零测集。做测度传递题时要看映射的正则性有多强。
44. 紧支撑连续函数 C_c 在很多 Lp 空间里都是稠密的，这使“先对光滑函数证明，再逼近推广”成为标准套路。没有稠密性，这条路线就走不通。
45. 卷积常能把粗糙函数平滑化。近似恒等核是把一般 Lp 函数逼近成光滑函数的重要工具。
46. 上极限 limsup 和下极限 liminf 对函数列、集合列都很有用。很多收敛题先写成 limsup/liminf 会更清楚。
47. 若级数一致收敛，则可以逐项积分；若导数级数一致收敛并控制一个基点值，则常可逐项求导。逐项运算前，先判断收敛方式。
48. 紧性、完备性、全有界性在度量空间里彼此相关但不等价。尤其在一般空间中，“有界”离“紧”还差很远。
49. 几乎处处收敛的函数列不一定有好的积分行为，但若再加一致可积性，很多结论会恢复。Vitali 收敛思想本质上就在这里。
50. 做实变函数题时，先判断问题核心是“收敛、测度、积分、逼近”中的哪一种。路线一旦选错，后面再细算也很难救回来。

## 6. 泛函分析

1. 在线性赋范空间里，范数必须满足正定性、齐次性和三角不等式。只差一条都不能叫范数。
2. 线性算子“有界”与“连续”在赋范空间中是等价的。证明一个线性算子连续，通常只要证明它在 0 点连续即可。
3. 算子范数定义为 sup{||Tx|| : ||x||≤1}，也等于 sup{||Tx|| : ||x||=1}。实际估计时常先在单位球上做文章。
4. 闭子空间比一般子空间重要得多，因为很多极限过程只有在闭子空间里才留得住。闭子空间上的完备性也更容易继承。
5. Banach 空间的闭子空间仍是 Banach 空间。反过来，赋范空间若本身不完备，闭子空间也未必帮你补齐。
6. 有限维赋范空间一定完备，而且所有范数都等价。这个结论在无限维空间里一般失效。
7. 有限维子空间在赋范空间中总是闭的。做商空间或补空间题时，这条结论常能直接省掉一步。
8. 不同范数在有限维空间给出同一拓扑，但在无限维空间可能差别巨大。很多“显然连续”的判断只在有限维里成立。
9. 线性双射 T 若定义在 Banach 空间之间且有界，则 T^(-1) 自动有界。这个结论就是有界逆定理。
10. 开映射定理说明 Banach 空间之间的满射有界线性算子会把开集映成开集。很多“逆算子连续”其实都靠它。
11. 闭图像定理说：若线性算子图像在 X×Y 中闭，且 X、Y 都是 Banach 空间，则该算子有界。它常用于证明一个算子“看起来不明显连续，其实连续”。
12. 一族有界线性算子若点态有界，则往往一致有界，这就是一致有界原理。注意它要求定义域是 Banach 空间。
13. 一致有界原理的典型用途是“逐点估计不够时，升级成范数统一估计”。很多反例也恰好说明完备性不能少。
14. Hahn-Banach 定理允许把定义在子空间上的有界线性泛函等范数延拓到整个空间。它是对偶空间理论的起点之一。
15. 分离超平面定理本质上源于 Hahn-Banach。看到“把点和闭凸集分开”的题，通常要往这条线上想。
16. 对偶空间 X* 由所有有界线性泛函组成。研究 X 时，很多时候先看 X* 反而更容易。
17. Hilbert 空间里每个连续线性泛函都能唯一写成 x ↦ <x,y> 的形式。这个结论就是 Riesz 表示定理。
18. 在 Hilbert 空间中，正交补 M^⊥ 总是闭集，而且 (M^⊥)^⊥ 等于闭包 M̄。带闭包这一点最容易漏。
19. Hilbert 空间里，对任意闭子空间 M，每个向量都能唯一分解为 x = m + n，其中 m∈M，n∈M^⊥。这就是正交投影存在唯一性的来源。
20. 正交投影 P 满足 P^2 = P 且 ||P|| = 1（只要投影不是 0）。它是“最近点”问题的标准答案。
21. 在 Hilbert 空间中，闭凸集上的最优逼近点存在且唯一。若集合还是子空间，误差向量必与该子空间正交。
22. Bessel 不等式说明任意向量在正交归一族上的傅里叶系数平方和不超过向量范数平方。它是 Parseval 等式的前半步。
23. Parseval 等式成立当且仅当这组正交归一族是完备的。少了完备性，只能得到不等式。
24. 可分 Hilbert 空间存在可数正交归一基。于是很多问题都能转成 l2 里的坐标问题。
25. 范数收敛一定推出弱收敛，但弱收敛通常不推出范数收敛。无限维空间里这两种收敛差得很远。
26. 弱收敛序列一定有界。若一个序列连有界都做不到，就别指望它弱收敛。
27. 在 Hilbert 空间里，若 x_n 弱收敛到 x 且 ||x_n|| 收敛到 ||x||，则 x_n 强收敛到 x。这个结论常叫 Radon-Riesz 性质。
28. 弱拓扑比范数拓扑更弱，因此弱闭集不一定范数开，范数紧也不等于弱紧。做拓扑比较时方向别弄反。
29. Banach-Alaoglu 定理说明对偶空间单位球在弱*拓扑下紧。它是弱*方法能够工作的根基。
30. 反身空间里，闭有界集在弱拓扑下是弱紧的。Lp 空间在 1<p<∞ 时是最常见的反身例子。
31. Lp 空间在 1<p<∞ 时反身，而 L1 和 L∞ 一般不是。很多紧性与极小化结论恰恰在这两端失灵。
32. 对偶关系里，Lp 的对偶通常是 Lq，其中 1/p + 1/q = 1。这个结论需要适当测度空间条件，别忘了前提。
33. 商空间 X/M 的范数定义为元素到子空间 M 的距离。只有 M 闭时，这个商空间才是 Hausdorff 的好空间。
34. 紧算子把有界集映成相对紧集。别把“有界算子”误认为“紧算子”，无限维里二者差别非常大。
35. 恒等算子在无限维 Banach 空间上不是紧算子。这个结论是判断“空间是否无限维”的常用试纸。
36. 紧算子对弱收敛序列有强化作用：若 x_n 弱收敛到 x，则 Kx_n 强收敛到 Kx。它在积分算子题里特别好用。
37. 紧自伴算子在 Hilbert 空间上有离散谱分解，除 0 外的谱由特征值组成。很多积分方程就是靠这个谱理论求解。
38. 紧算子的非零特征值只能积聚到 0。若看到一串非零特征值积聚到别处，多半哪里设错了。
39. 一般有界算子的谱非空且紧，而且位于闭圆盘 {|λ|≤||T||} 内。这个“谱一定非空”是无限维线性代数里很特别的现象。
40. 谱半径满足 r(T) = lim ||T^n||^(1/n)。算不出谱时，它常给出很有用的估计。
41. 自伴算子的谱都落在实轴上，正算子的谱还会落在非负实轴。做 Hilbert 空间算子题时，伴随结构能大幅简化问题。
42. Hilbert 空间中 ||T*|| = ||T||，且 T 自伴当且仅当 <Tx,y> = <x,Ty>。伴随算子是检查对称性和能量估计的核心工具。
43. 稠密子集上定义的有界线性算子至多只有一个连续延拓。先在好函数上做，再延拓到全空间，是泛函分析中的常见套路。
44. 闭单位球在赋范空间中范数紧，当且仅当空间有限维。这个结论和 Heine-Borel 在无限维失效是同一件事。
45. Riesz 引理常用于构造“彼此分离”的单位向量列，从而证明某些集合不紧。它是无限维反例制造机。
46. 一列有界线性算子若在每个点都收敛到某个极限，且定义域是 Banach 空间，则极限算子仍有界。这个结论通常借助一致有界原理。
47. 双线性型连续，当且仅当它对两个变量有统一有界估计 |B(x,y)| ≤ C||x||||y||。在 Hilbert 空间里，这常对应一个有界算子。
48. Lax-Milgram 定理说明强制有界双线性型对应的变分问题有唯一解。椭圆型偏微分方程弱解理论大量依赖它。
49. Fredholm alternative 告诉我们“解存在”与“齐次方程零空间”之间有互补关系。它是紧算子方程理论的核心现象之一。
50. 做泛函分析题时，先认清对象是“空间、拓扑、算子、对偶、紧性”中的哪一类最关键。无限维里最危险的误区，就是把有限维直觉直接照搬过来。

## 7. 抽象代数

1. 判断非空子集 H 是否为子群，常用一条线检验：若对任意 a,b∈H 都有 ab^(-1)∈H，则 H 是子群。这个判据比逐条验更省事。
2. 循环群的所有子群仍是循环群。有限循环群里，每个子群都由某个因子对应。
3. 有限群中元素的阶一定整除群的阶，这是 Lagrange 定理的直接推论。易错点：反过来“每个整除数都对应元素阶”并不总成立。
4. 阶为素数的群一定是循环群，也一定是阿贝尔群。这是最简单却最常用的结构判别之一。
5. 指数为 2 的子群一定正规。因为左陪集和右陪集都只能剩下“自己”和“补集”两种。
6. 中心 Z(G) 总是正规子群。它往往是判断群“离交换还差多远”的第一站。
7. 只有正规子群才能拿来构造商群。非正规子群写出的陪集集合通常没有自然群结构。
8. 群同态的核一定是正规子群，像一定是子群。很多结构题先找核和像，问题就会立刻清楚。
9. 第一同构定理是群论最常用工具之一：G/kerφ 同构于 Imφ。做商群题时，先想“它是不是某个同态的像”。
10. 两个子群的乘积 HK 不一定是子群。若 H、K 之一正规，或 HK = KH，结论才更稳。
11. 共轭是群里最自然的“对称关系”，共轭类把群分块。正规子群的本质就是对共轭封闭。
12. p 群的中心非平凡。这个结论常用来做归纳、证明有限 p 群含正规子群。
13. 元素阶等于其生成循环子群的大小。做具体计算时，“求元素阶”和“求最小正整数 n 使 g^n=e”等价。
14. 在直积 G×H 中，元素 (g,h) 的阶是 ord(g) 与 ord(h) 的最小公倍数。不是简单相乘，除非两者互素。
15. 群作用把“群元素”转成“集合上的变换”。很多计数问题一旦写成群作用，就能直接用轨道语言处理。
16. 轨道-稳定子定理给出 |Orb(x)| = [G:Stab(x)]。它把“元素能被移到多少地方”和“有多少对称性保留它”联系起来。
17. Burnside 引理说轨道数等于平均不动点数。遇到“考虑对称后本质不同有多少种”时，这几乎是标准答案。
18. Sylow 定理给出 p-Sylow 子群的存在、共轭性和个数限制。数 Sylow 子群常常是判定群结构的关键一步。
19. Sylow p 子群若唯一，就必正规。很多“证明正规”题其实就是在证明 Sylow 个数只能取 1。
20. 交换化 G/[G,G] 是群最自然的阿贝尔商。它把“所有交换子都压成单位元”，保留了最粗的交换信息。
21. 环同态保加法、乘法并通常保 1（若题目约定如此）。核是理想，像是子环，这和群同态非常平行。
22. 整环里没有零因子，因此 ab=0 推出 a=0 或 b=0。很多多项式次数公式和消去律都依赖这个性质。
23. 有限整环一定是域。因为乘以非零元给出单射，而有限集上单射就是满射。
24. 域中的理想只有 {0} 和整个域。反过来，一个交换环若只有这两个理想，它就是域。
25. 理想比子环更适合做商环，因为只有理想才能保证乘法良定义。做商结构时先分清“子环”和“理想”。
26. 第一同构定理对环同样成立：R/kerφ 同构于 Imφ。遇到商环题时，先试着构造自然同态。
27. 素理想对应的商环是整环，极大理想对应的商环是域。极大一定素，但素不一定极大。
28. 在 Z_n 中，可逆元恰好是与 n 互素的剩余类。做单位群题时，先看 gcd(a,n)。
29. 中国剩余定理把互素模数下的同余系统拆开再拼回。它既能解方程，也能分解环结构。
30. 主理想整环 PID 一定是唯一分解整环 UFD，但反过来不一定成立。层次关系别记反。
31. 在 UFD 中，不可约元和素元等价；在一般整环里这两者可能不同。做因式分解题时要先看所在环够不够好。
32. Euclid 整环一定是 PID，因此也一定是 UFD。F[x] 和 Z 都是最典型的例子。
33. 多项式除法要求除式首项系数可逆，最常见情形是系数来自域。换到一般环里，这一步未必能做。
34. 域上 n 次多项式至多有 n 个根。这个结论在整环上也成立，但有零因子的环里可能失效。
35. 因式定理说 f(a)=0 当且仅当 x-a 整除 f(x)。找根和拆因子本质上是同一件事。
36. F[x] 在域 F 上是 PID，所以极大理想都由不可约多项式生成。商环 F[x]/(p(x)) 常用来构造域扩张。
37. Eisenstein 判别法是判不可约的高效充分条件，但不是必要条件。用不出来不代表多项式可约。
38. Gauss 引理说明整系数原始多项式在 Q[x] 不可约，当且仅当它在 Z[x] 不可约。它把有理系数问题拉回整数系数。
39. 域扩张 E/F 若有限，则 E 是 F 上的有限维向量空间。扩张次数就是这个维数。
40. 代数元的最小多项式一定首一且不可约，并且整除所有能把该元素消掉的多项式。它是研究代数元的核心对象。
41. 域同构会保持最小多项式和代数关系。很多“根怎么对应”的问题，本质上都在追踪这个结构不变量。
42. 分裂域总存在，而且在同构意义下唯一。研究一个多项式的全部根，最自然的背景就是它的分裂域。
43. 有限域的大小一定是 p^n，其中 p 是素数。反过来，每个 p^n 都存在且在同构意义下唯一。
44. 有限域的乘法群一定是循环群。也就是说，F_q^× 总能由一个本原元生成。
45. 在有限域 F_q 中，每个元素都满足 x^q = x。于是多项式 x^q - x 的根恰好是 F_q 的全部元素。
46. 特征为 0 的整环包含一个同构于 Z 的子环，特征为正的整环其特征一定是素数。特征不可能是合数。
47. 模与向量空间很像，但系数环里元素未必可逆，因此很多线性代数直觉不能直接搬来。比如子模不一定有补。
48. 幂零元一定不是单位元（在非零含幺环中）。因为若 a^n=0 且 a 可逆，就会推出 1=0。
49. 幂等元 e^2=e 常对应环的直积分解。看到非平凡幂等元时，往往意味着这个环可以拆块。
50. 做抽象代数题时，先认清对象是“群、环、域、模、扩张”中的哪一类，再选工具。很多难题一旦找到合适同态或商结构，计算会突然变简单。
